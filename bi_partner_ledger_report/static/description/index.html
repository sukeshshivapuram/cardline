Features

Different Partner Reports.

Provide Aged Partner Report as well as Partner Ledger Report in odoo 12.

Print Report.

Print Aged Partner Report and Partner Ledger Report in PDF and Excel format.

Date and Other Filter Option.

Provide Date Range filter, Reconciled Entries, Period Length filter on wizard for partner ledger and aged partner reports.

Journal Filter on Report.

Print Different Journal's Report using Journal Filter, User can select multiple journals also and print selected journals reports in group.
